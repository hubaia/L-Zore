# 证据清单

## 📋 证据系统概述

### 🎯 证据设计原则
1. **真实性**：每个证据都有其存在的合理性
2. **关联性**：证据之间相互关联，形成证据链
3. **层次性**：从表面证据到深层证据，逐步揭示
4. **多样性**：包含各种类型的证据，增加游戏丰富性

### 📊 证据分类
- **物理证据**：实体物品和文件
- **数字证据**：电子文件和记录
- **证言证据**：角色口述和记录
- **环境证据**：现场环境和痕迹
- **时间证据**：时间记录和序列
