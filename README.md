# 🎮 Vite + Phaser + React UI 卡牌游戏

这是一个使用现代技术栈构建的高性能21点卡牌游戏项目。

## 🚀 技术栈

- **⚡ Vite** - 极速构建工具和开发服务器
- **🎮 Phaser 3** - 强大的2D游戏引擎
- **📦 phaser-react-ui** - Phaser与React的无缝集成
- **⚛️ React 19** - 最新版本的React框架
- **🛣️ React Router v7** - 现代路由系统
- **🎨 Tailwind CSS** - 原子化CSS框架
- **📘 TypeScript** - 类型安全

## ✨ 项目特色

### ⚡ Vite优势
- 极速热重载 (HMR)
- TypeScript原生支持
- 现代ES模块
- 优化的构建过程

### 🎮 游戏特性
- React + Phaser集成
- 响应式缩放
- 流畅动画效果
- 事件驱动架构

### 🃏 游戏功能
- 完整的21点游戏规则
- 智能A牌计分（1或11点）
- 庄家自动AI（17点以下要牌）
- 精美的卡牌发放动画
- 实时胜率统计
- 游戏状态持久化

## 🎯 游戏规则

- **目标**：让手牌总分接近21点而不超过
- **计分**：
  - A可以是1点或11点（自动选择最优）
  - J、Q、K都是10点
  - 其他牌按面值计分
- **规则**：
  - 超过21点即爆牌
  - 庄家17点以下必须要牌
  - 先到21点或更接近21点者获胜
